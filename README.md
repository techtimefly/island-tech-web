# Island Tech Web

Static website source for Island Tech IO and Mobile Meridian Services.

## Sites

- `sites/islandtech/` -> Island Tech IO (`islandtech.io`)
- `sites/mobilemeridian/` -> Mobile Meridian Services (`mobilemeridianco.com`)

Each folder is a standalone static site that can be deployed independently to GitHub Pages.

## Local Preview

From this folder:

```bash
npm run serve:islandtech
npm run serve:mobilemeridian
```

Then open:

- `http://localhost:4173` for Island Tech IO
- `http://localhost:4174` for Mobile Meridian Services
